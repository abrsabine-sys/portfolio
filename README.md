## 🚀 Featured Projects

### 🎓 Campus Connect
Campus Connect is a full-stack web application designed to streamline event management at universities. Users can:

Register accounts using email, username, and password

Join their school

Create, view, and delete events

RSVP to events or cancel attendance

See real-time attendee counts for events

The application is built with the following technologies:

Backend: Node.js, Express, RESTful API

Frontend: HTML, CSS, JavaScript (vanilla)

Data Management: In-memory arrays for users and events

State Handling: LocalStorage for session persistence

Key Features & Skills Demonstrated:

Full-stack development from frontend to backend

CRUD operations for events and users

Authentication flow and session management

Dynamic DOM updates and real-time event filtering

Error handling and API response validation
- Users can register, join a school, create events, RSVP, and manage attendance
- **Tech:** Node.js, Express, REST APIs, HTML, CSS, JavaScript
